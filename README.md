## Описание программы
GUI-приложение "Weather Diary" позволяет добавлять, просматривать и фильтровать записи о погоде. Данные сохраняются в файл JSON. Приложение включает базовую валидацию ввода и интегрировано с Git для управления версиями.

## Как запустить приложение
1. Убедитесь, что у вас установлен Python.
2. Установите Tkinter (обычно входит в стандартную установку Python).
3. Сохраните код в файле, например, `weather_app.py`.
